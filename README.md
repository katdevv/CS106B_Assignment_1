# CS106B Assignment 1

This is the first assignment of the **CS106B: Programming Abstractions** course from Stanford.

This assignment consists of the following problems:
- **Warmup** - assigns a hash code to your name.
- **Consecutive Heads** - simulates flipping a coin until three consecutive heads are tossed. 
- **Combinations** - generates Pascal's Triangle 
- **Numeric Conversions** - converts integer to sting and string to integer
- **Flesch–Kincaid Readability** - calculates input text's reading grade-level
